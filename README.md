# ⭐ Social-Media-App
* Demo: https://social-media-app0.web.app
* A small and simple social media app. You can have a profile, upload a profile photo, can share a post, like a post and comment on a post with this application

# 🚀 Tech Stack

* Node.js
* Tailwindcss
